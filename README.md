# Text To Speech

A C# application that converts written text into spoken audio using text-to-speech synthesis.

## Features

- **Text Conversion**: Convert any text input into natural-sounding speech
- **Audio Output**: Generate and play audio files
- **C# Implementation**: Built with C# for robust and efficient performance
- **Cross-Platform Support**: Run on Windows, Linux, and macOS

## Getting Started

### Prerequisites

- .NET Framework or .NET Core installed
- Visual Studio or any C# IDE

### Installation

1. Clone the repository:
```bash
git clone https://github.com/GayathriHubb/Repo46-TextToSpeech.git
```

2. Navigate to the project directory:
```bash
cd Repo46-TextToSpeech
```

3. Build the project:
```bash
dotnet build
```

### Usage

Run the application and provide your text input to generate speech:
```bash
dotnet run
```

Enter the text you want to convert to speech, and the application will process and output the audio.
